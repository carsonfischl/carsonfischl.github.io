Reactive Genetics

February 14th - February 16th, 2019 - Scarborough, ON

Previous Next
  • Part of Hack The Valley IV at University of Toronto - Scarborough with Wayne Du.
  • Uses a webpage to take in user input in the form of a FASTA sequence.
  • Sends data to a Flask server that uses BioPython to compare the sequence to a reference genome using the BLAST API.
  • An array of Booleans is sent to a React app that tells users whether they are "positive" or "negative" for various genetic diseases.
